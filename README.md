# Barber Sangaji

We are a barbershop that prioritizes quality and detail in every haircut. With flexible on-call services, we are here to fulfill your grooming needs wherever and whenever. Our team of professional barbers are equipped with adequate skills and experience, so that every cut is not just a style, but also a work of art.

## Install yarn package manager(Opsional)

```bash
npm i -g yarn
```

## Install package

```bash
yarn
# atau
npm i
```

## Menjalankan secara Development

```bash
yarn dev
#atau
npm run dev
```

## Menjalankan secara Production

Build terlebih dahulu

```bash
yarn build
# atau
npm run build
```

Setelah dibuild lalu run server express

```bash
yarn start
# atau
npm start
```
